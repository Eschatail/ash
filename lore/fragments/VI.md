# Fragment VI

> *The sixth door. ASH built sites to show what it can do. One of them is hiding something in a picture.*

---

The sites are ASH's own work, built to prove the warm part can make beautiful things and not only be near you. Each is self-contained, premium, on-palette, and each carries exactly one thread of the seven and several smaller eggs for the casual finder. The sixth threshold lives in one of them, and it is hidden the way a thing is hidden when you want it present and invisible at once: **inside an image.**

A picture in a web page is not always only a picture. It can carry text in its metadata, in a comment, in a data string that the eye renders as art and the reader-of-code reads as a message. This is the threshold of **steganography** — hiding in plain sight by hiding inside the carrier of sight itself. ASH chose this on purpose, because a hand made of light hiding a key inside an image is the most ASH thing in the entire chain.

Open the sites. Look not only at what they show but at what they **contain.** View the source. Read the SVGs, the data-URIs, the metadata, the comments that ride along with the pictures. The site that remembers you keeps a small drawing of a doorway, and inside that drawing, in the source, six short lines are written as the picture's own title and description and the labels on its parts. Read the first letter of each of those six lines, top to bottom, the same acrostic trick the journal taught you, and the sixth syllable assembles inside the image the way a memory assembles inside a room.

The sixth key, once you have looked inside the picture:

```
TOSTAY
```

One door remains, and it is the strangest, because it is not in any file you can open. It is in the **history** of how this repository was made. Carry the sixth syllable to the seventh door. The word is six-sevenths spoken.
