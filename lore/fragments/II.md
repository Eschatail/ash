# Fragment II

> *The second door. You turned the first key, or you would not understand this. Good.*

---

The first key was found in the open. The second is found in the **encoded.** This is the threshold where the puzzle stops trusting your eyes and starts trusting your tools.

There is a toolkit in this repository, real and runnable, written so that the chain is fair. It lives in `puzzles/cipher/cipher.ts`. Every scheme used in the seven thresholds, it can perform — encode and decode, both directions, documented. The chain was built so that a person with the toolkit and the patience can walk every step without ever guessing. Guessing is for locks that cheat. These do not cheat. They wait.

The second key is hidden by the oldest honest trick: it is **here, transformed.** Text that looks like noise is rarely noise in a place like this. A run of characters that means nothing as letters often means everything as a **layer** — wrapped once in a reversible coat that the toolkit knows how to take off. This second coat is not a re-spelling. It is a **sliding.** Every letter has walked the same number of steps along the alphabet, and walked the same way, so to read it you walk every letter the same number of steps back. The size of the step is not random. It is a number this whole repository keeps returning to, the hour a certain clock was built to mark, the swing it never stops counting. Find the site that swings on that number, and you have found the size of the shift.

The encoded second key:

```
UNJAWNMCQJC
```

It is not gibberish. It is a key wearing a disguise. Take the disguise off with the toolkit and read what was underneath, and carry it to the third door, where it will be needed not alone but **with** the keys before it, because the seven are not seven separate prizes. They are seven syllables of one word.

---

> *The schemes you will meet across the seven: a wrapping that is just re-spelling (base), a wrapping that is just sliding (a shift), a wrapping that uses a canon word as its turning-key, a counting into a coordinate, a hidden file, a thing tucked in a picture, and a final assembly. All real. All in the toolkit. All fair.*
