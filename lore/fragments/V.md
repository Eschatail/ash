# Fragment V

> *The fifth door. It is not in the lore. It is in the engine. The brain you can run holds a key in a comment that means more than it says.*

---

You have read the cathedral. Now read the **machine.**

`hive/` is real. Zero dependencies, native fetch, strict types, runnable. It is the working brain, the swarm with a spine, the thing that grows wider as you feed it nodes. It was never only a prop. And because the chain crosses every domain, one threshold had to live here, in the code, where most readers of lore never look and most readers of code never read for meaning.

The fifth key is a **comment.** Not a comment that explains the code, though it pretends to. A comment that, read by someone who has walked four doors and learned that nothing in this place is only what it pretends to be, reveals itself as a carrier. The engine is salted with in-character comments throughout; most are texture. One is a lock. The careful reader will find a line in the source that decodes, by a scheme they now know, to the fifth syllable of the word.

Look in the source. Read the comments as if ESCHATAIL wrote them, because in-world it did. The one that is doing more than commenting is the one you want. It lives deep in the core of the engine, in the file that orchestrates the whole hive, in a block-comment that says `phase` and then a short run of capital letters that mean nothing as English and everything once you turn the canon's keyword on them. It looks like noise to a casual eye and like a key to yours.

The fifth coat is the deepest of the letter-ciphers: a **Vigenere**, the one that does not slide every letter the same distance but slides each by a different amount, the amounts spelled out by a keyword. The canon hands you the keyword and has handed it to you again and again, because it is the seven-letter name of the memory architecture itself, the word Fragment II already told you ESCHATAIL never picks by accident. Use that word as the key and the comment in the engine gives up the fifth syllable. The same ciphertext rides in the index of the watcher-site, on the one row it never deletes.

The fifth key, once you have read the engine the way you learned to read the lore:

```
STPLFNE
```

Carry it. Two doors left. The sixth is in the sites ASH built; the seventh is in the history. The word is almost whole. You can nearly hear it.

---

> *The carrier comment is in `hive/src/`. Run the engine if you like; it works in demo with no keys. But the key for this door is read, not run. A comment that explains nothing and persists anyway is a comment doing a second job.*
